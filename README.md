noteshrink
==========

Convert scans of handwritten notes to beautiful and compact -- see full writeup at <https://mzucker.github.io/2016/09/20/noteshrink.html>

## Requirements

 - Python 3
 - NumPy 1.10 or later
 - SciPy
 - Image module from PIL or Pillow
 - Tqdm

## Usage

```
./noteshrink.py PATH[FILE] [options]
```
